TSP Library Tester: Instructions
 
Description
 
The ISR TSP Library controls TSP-based devices. This program tests theLibrary.
 
Connecting to an instrument
  • Start the TSP Library Tester from Start, Programs, ISR, TSP Library 3.0.
  • The tester scans and displays the address of available resources. 
  • If the tester finds resources it enables the Connect button:
  • Select the resource address for the desired instrument.
  • Click Connect to connect to the instrument.
  • Connect one or more TSP instrument and click Refresh VISA.
  • The connect button should be enabled allowing you to connect.
 
TSP Library Testing using a 26xx Instrument
 
Entering Commands from the Console Tab
The console tab allows entering commands and getting a response back. Commands entered in the Instrument Input edit box are sent to the instrument upon hitting the Enter key. Responses are displayed in Instrument Output panel.  The responses are terminated by the TSP prompt depending on the status of the Enable Prompts check box located under the Instrument tab.
  • Select the Console tab;
  • Type the command
    print(_VERSION)
    in the Instrument Input text box or place the cursor at the end of the command already in the edit box;
  • Press enter.
  • The output text box will display the response from the instrument:
    Lua 5.0.2
  • For other commands, check the instrument User's Guide. 
 
Loading and Executing functions
  • Select the Functions tab.
  • A TSP Script function calculating the sum of its arguments is already listed in the Function Code edit box. 
  • Click Load Function:
  • This loads the function script into the instrument.  The message Function Loaded will be displayed in the status bar.
  • Click CALL:
  • This executes the function which name is specified in the Function Name text box. The CALL supplies to the function the argument values from the Function Arguments edit box. 
  • The message 'Function Called Successfully' will be displayed in the status bar.
  • The result of the calculation is displayed under the Console tab.
  • The output from the instrument is displayed in the Instrument Output panel under the Console tab:
  • Click Console to open the console tab. 
  • The output from the instrument is displayed.
  • Below the result is the outcome of the Call command - true.
  • Enter a different set of argument and repeat the CALL command.
 
Loading and Running Scripts
  • Select the Scripts tab.
  • Click the ellipsis and browse and select the product.tsp file located under the tester Scripts folder, which is a sibling folder to the tester program folder. 
  • This imports the script to the Script edit box and update the script name in the Script Name edit box.
  • Click Load.
  • This loads the script to the instrument.
  • The script name will be added to the User Scripts list box.
  • Click Run. 
  • This compiles the script making its functions known to the instrument.
  • Or, click Load and Run to do both at once.
  • Select the Function tab.
  • Enter 'product' as the function name (the name is case sensitive).
  • Enter 2 or more arguments separated with commas.
  • Click CALL. 
  • This executes the function.
  • Click Console to open the console tab. 
  • The result of the calculation is displayed in the Instrument Output panel.
