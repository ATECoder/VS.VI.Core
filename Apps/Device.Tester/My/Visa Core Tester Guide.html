Description






The ISR VISA Library controls VISA instruments. This program tests the Core Library. Testing can be accomplished using three programs:


  • Simple Read and Write;
    • 

  • Basic GPIB Test Panel; and 
    • 

  • VISA Events Service Requester.
    • 

    




  









Simple Read and Write











Description






This panel the allows sending (writing) and receiving (reading) messages from message-based VISA instruments over the IEEE488 (GPIB), Serial (ASRL), or Ethernet bus. 



  









Open






Select Simple Read and Write from SWITCHBOARD and click Open.  This opens the  Simple Read/Write panel.  



  









Open Session






Click Open Session. This opens the Select Resources dialog displaying all the resources available locally.  Instruments have VISA resource names ending with INSTR.  For example, the resource string for an instrument addressed at 24 on board 0 is GPIB0:24:INSTR. 


 


Select an instrument from the list and click OK.


 


If the instrument does not appear as part of the list of resources, try entering the resource (e.g., GPIB0:24:INSTR) manually and click Open Session.  Close and re-open the session and see if the resource is now listed.  Another quirk of NI VISA.


 



  









Close Session






Click Close Session to close the session.



  









Commands






Once opened, access to the instrument is enabled and the Query, Write, and Read buttons can be used to exchange messages with the instrument.







Identity






Send the Identity (*IDN?) command to get the instrument manufacture and version information:


  • Enter the Identity command *IDN?\n in the Message to Send box. 
    
    The '\n' signifies the termination character. The program replaces this with the termination character specified for the instrument, typically a Line Feed (value 10).
    • 

  • Click Query to send the message and read back the result form the instrument.  The result is displayed in the Message box and the time in the ms box.
    • 

  • Alternatively, click Write and then Read.
    • 




  





  









Clear






Clear Clear to clear the Messages box. 



  









Execution Times






Execution times can be measured for query, write or read operations. The execution time is displayed in the ms text box.



  





  









Basic GPIB Test Panel






 


 







Open






Select Basic GPIB Test Panel from the SWITCHBOARD and click Open.  This opens the GPIB Test Panel.  



  









Connect






  • Enter the Primary Address (e.g., 26) from the CONNECT tab;
    • 

  • Depress the CONNECT button;
    • 

  • Select the SEND/RECEIVE tab.
    • 

    




  









Send/Receive






Use the send receive tab to communicate with instruments using the GPIB (IEEE 488) Bus.  Instruments could be:


  • SCPI instruments
    
    Instruments using the SCPI protocol; or 
    • 

  • TSP Instruments
    
    Instruments using the Keithley TSP protocol.
    • 








SCPI instruments






Example: 


  • Select *IDN? from the SINGLE COMMANDS drop down;
    • 

  • Click SEND COMMAND;
    • 

  • Click READ SRQ (enable receive). This enables the receive button if there is data in the instrument output buffer as indicated also by the status byte displayed on the status bar;
    • 

  • Click RECEIVE. This reads the message from the instrument and displays it in the RECEIVED FROM INSTRUMENT text box.
    • 

  • Click DISCONNECT from the CONNECT tab releasing the button and disconnecting the instrument.
    • 




  









TSP Instruments






  • Check Send Commands On Disconnect from the Connect tab and make sure the command localnode.prompts = 0 is listed in the command box below.
    • 

  • Open the SEND/RECEIVE tab;
    • 

  • Select *IDN? from the SINGLE COMMANDS drop down;
    • 

  • Click SEND COMMAND;
    • 

  • Click READ SRQ (enable receive). This enables the receive button if there is data in the instrument output buffer as indicated also by the status byte displayed on the status bar;
    • 

  • Click RECEIVE. This reads the message from the instrument and displays it in the RECEIVED FROM INSTRUMENT text box.
    • 

  • Select print( _VERSION ) the SINGLE COMMANDS drop down;
    • 

  • Click SEND COMMAND;
    • 

  • Click READ SRQ;
    • 

  • Click RECEIVE;
    • 

  • This displays the LUA version stored in the instrument.
    • 

  • Click DISCONNECT from the CONNECT tab releasing the button and disconnecting the instrument.
    • 

  • Click CONNECT while looking at the instrument panel.  If no errors are displayed, the disconnection commands executed properly.
    • 

  • Click DISCONNECT.
    • 




  





  





  









VISA Events Service Requester






The VISA Events Service Requester panel automatically displays messages read from the instrument using the event requester call backs supported by the VISA protocol standard. 







Description







  









Open






Select VISA Events Service Requester from the SWITCHBOARD and click Open.  This opens the VISA Events Service Requester.  


 


Follow the instructions on the panel to test the event requester.
